# dev-utils-kit
A lightweight and practical utility library for front-end and back-end developers.

## Introduction
This project collects commonly used general functions in daily development, including date formatting, function throttling/debouncing, data processing and other universal tools. It is designed to reduce repetitive code writing and improve development efficiency.

## Usage
```javascript
const utils = require('./index');

// Format date
console.log(utils.formatDate(Date.now()));
